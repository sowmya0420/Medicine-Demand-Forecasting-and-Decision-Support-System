# Medicine-Demand-Forecasting-and-Decision-Support-System
This project introduces a Medicine Demand Forecasting and Decision Support System that is developed as a system to support stock management in the pharmacy, using soft computing techniques. 
The system incorporates four interrelated modules, which include: 
  1. fuzzy logic-based medicine movement classification
  2. demand forecasting
  3. intelligent alert generation 
  4. expiry risk optimization. 

The first module uses fuzzy membership functions with sales data (in terms of the total units sold, number of transactions and average units sold) to identify the medicines as Fast, Moderate or Slow moving. 

The second module uses fuzzy inference to forecast the demand in the coming month and determine the status of the stock as
overstocked, understocked, or adequate. 

The third module creates intelligent alerts in real-time of the understocked and expiring medicines at store locations.

The fourth module calculates the fuzzy expiry risk score using days to expiry, the level of demand, and stockquantity to make the best inventory decisions. 

Overall built as an interactive Streamlit web application and tested onsynthetic pharmacy data, the system demonstrates
effective application of fuzzy logic in healthcare supply chain management.
